# Bug Battle

Bug Battle is a minimal web-based game where players fix bugs in code snippets under a time limit.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

3. Open your browser to:

```
http://localhost:3000
```

## How to play

- Click **Start Game** to load a question.
- Fix the bug in the textarea.
- Submit before the timer ends.
- Track your score and save it to the leaderboard.
